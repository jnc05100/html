설명class : 여러 개의 태그에 함께 사용할 수 있다. 같은 디자인을 여러 곳에 줄 때 쓴다. id : 한 페이지에서 하나만 사용하는 것이 원칙이다. 특정 요소 하나를 지정하거나 JavaScript에서 많이 사용한다. 여러 요소에 같은 스타일·동작을 주고 싶으면 class, 페이지에서 단 하나뿐인 요소를 특정하고 싶으면 id를 사용한다.

<p class="text">반갑습니다</p>

<h1 id="title">제목</h1>

<form action="/login" method="POST">

  <label>아이디</label>
  <input
    type="text"
    name="username"
    placeholder="아이디">

  <br><br>

  <label>비밀번호</label>
  <input
    type="password"
    name="password"
    placeholder="비밀번호">

  <br><br>

  <button type="submit">로그인</button>

</form>

여기서 name이 꼭 필요한 이유: name이 있어야 서버가 "이 값이 어떤 항목인지" 구분할 수 있다. name이 없으면 값은 입력해도 서버로 아예 전달되지 않는다. 위 form을 submit하면 action="/login" 주소로, name을 key로 삼아 아래처럼 데이터가 서버로 전송된다.

password=1234
